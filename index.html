Welcome!

Dunst is a minimalistic notification daemon for Linux. It's designed to fit nicely into minimalistic windowmanagers like dwm, but it should work on any Linux desktop.

Features

  • highly configurable
    • font (xft-fonts support)
    • colors
    • timeouts
    • screenposition
    • message format
    • etc.
  • keyboard shortcut to remove messages
  • history (redisplay old messages via keybinding)
  • custom rules to override colors, timeouts etc. of certain messages

Installation

Arch Linux

There is a package in the aur called dunst-git

Installing manually

Dependencies

  • dbus
  • libxinerama
  • libxft
  • libxss
  • iniparser
$ git clone https://github.com/knopwob/dunst.git
$ cd dunst
$ make
$ sudo make PREFIX=/usr install

Configuration

An example configuration is provided as /usr/share/dunst/dunstrc. Copy this to $XDG_CONFIG_HOME/dunstrc (this normally translates to ~/.config/dunstrc) and edit it to your liking.
